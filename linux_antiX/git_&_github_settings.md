# **Установка и настройка `Git` и `Git Hub` на ОС `antiX` `Linux`**

## Описание
Данна памятка призвана помочь начинающему пользователю использовать систему контроля версий Git на своем компьютере и загружать свои наработки в удаленный репозиторий на Git Hub.
