# VS CODE
```bash
code .                        # Текущая папка (если установлен)
code ~/Documents              # Папка Documents
sudo apt install code-oss     # Установка (если нужно)
Ctrl+Shift+` # open new terminal in vscode
Ctrl+Shift+P # show all commands (print open settings json)
Ctrl+Shift+E # open explorer
Ctrl+Shift+G # open source control
Ctrl+Shift+X # open extensions
```
